# Proyecto: Portfolio personal

## Objetivo
Portfolio personal para mostrar proyectos, especialmente Alebet Manager, y mejorar empleabilidad como Fullstack Developer.

## Estado actual
Proyecto viejo actualizado parcialmente en tecnologias.
La base funcional existe, pero necesita mejoras de UX/UI, estructura de proyectos y narrativa profesional.

## Stack actual
- Next.js 16
- React 19
- Tailwind CSS 4
- TypeScript 5
- Framer Motion
- Zod
- next-themes

## Problemas actuales
- La seccion de proyectos no comunica suficiente profundidad
- Falta una vista rapida con modal
- Falta una pagina dedicada para profundizar en cada proyecto
- Las transiciones son basicas o poco visibles
- La experiencia debe ser mobile first

## Objetivo de esta iteracion
Implementar una UX de proyectos con:
- modal de vista rapida
- pagina dedicada por proyecto
- transiciones mas suaves y visibles
- enfoque mobile first

## Proyecto principal
Alebet Manager:
sistema fullstack para gestion interna de laboratorio, con foco en stock, movimientos y trazabilidad.

## Reglas
- No romper estructura existente si no es necesario
- Priorizar componentes reutilizables
- Priorizar mobile first
- Mantener diseno limpio y profesional
- Evitar sobrecarga visual
